EVEE-Tools



EVEE-tools is a suite of scripts used in Chen et al., 2017 to analyze expression evolution across species. To begin, download all scripts and example data from the github repository

You may find these scripts helpful for the following tasks:



./makeExpressionMatrix.py
usage: makeExpressionMatrix.py [-h] [--gene_name GENE_NAME] ortholog_groups rsem_list out_prefix

This simple script generates a matrix files (gene x sample) of TPM values given a file of ortholog groups and a list of RSEM output files.

Input:

  • List of ortholog groups. Each line should begin with a unique ortholog ID and be followed by the gene ID (e.g. Ensembl ID) from each species that belong in that ortholog group. Each gene ID should also be prefixed by a species ID. The gene ID should match the gene IDs in the RSEM output files.
  • List of RSEM output files. Each line should list the species, species ID, and path to corresponding RSEM output files. Multiple output RSEM files belonging to the species can be listed.
  • (Optionally) Mapping from gene ID to gene name.

Output:
  • An expression matrix of TPM values per gene per sample.
  • An index file specifying which species belongs to each column of the matrix file.

Example:
./makeExpressionMatrix.py --gene_name data/gene_name.txt data/mammals.ortholog_groups.txt data/liver.rsem.list liver
